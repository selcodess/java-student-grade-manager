# Java Student Grade Manager

A simple Java console application created to practice basic Object-Oriented Programming concepts.

This project manages a small list of students, stores their grades, checks whether each student passed or failed, and calculates a class summary.

## Features

- Create student objects
- Store students in an `ArrayList`
- Print student information
- Check if a student passed or failed
- Calculate average grade
- Count passed students
- Count failed students
- Validate grade values between 0 and 100

- Constructors

- Methods
- If / else statements
- ArrayList
